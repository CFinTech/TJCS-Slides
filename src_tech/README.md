## Principle

- 这里展示的素材，与科研场合（eg., 组会）并不匹配，**<u>不建议</u>**在任何科研相关场合下使用本素材
- 这里的素材适用于在本科阶段任何有关科技、专业知识相关的 slides 中
- 这里的素材并非个人制作，而是从各个渠道收集的
