## Principle

- 这里的素材适用于思政课的 presentation 以及主题介绍
- 当然，对于思政课的汇报来说，通常做好 silides 是不够的，我建议补充一些视频、问卷数据等
- 这里的素材并非个人制作，而是从各个渠道收集的
